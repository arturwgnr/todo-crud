# 📝 Task Organizer

Aplicação de lista de tarefas com JavaScript puro + LocalStorage + Design Responsivo.

## Funcionalidades
- Adicionar tarefas
- Marcar como concluídas
- Editar tarefa existente
- Excluir tarefas
- Armazenamento local (LocalStorage)

## Tecnologias
- HTML
- CSS
- JavaScript
