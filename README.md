# 📝 Task Organizer

A simple task list application built with vanilla JavaScript, LocalStorage and responsive design.

## Features
- Add new tasks
- Mark tasks as completed
- Edit existing tasks
- Delete tasks
- Persistent storage using LocalStorage

- CSS
- JavaScript
